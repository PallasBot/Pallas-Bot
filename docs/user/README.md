# 使用指南

面向**部署者与群管理员**：如何跑起来牛牛、在网页里改配置、常用口令与排障。开发与插件编写见 [开发指南](../develop/README.md)。

## 快速开始

| 步骤 | 文档 |
| --- | --- |
| 1. 准备环境与配置 | [配置要点](../Config.md) |
| 2. 安装并启动 | [标准部署](../Deployment.md) 或 [Docker](../DockerDeployment.md) |
| 3. 打开网页控制台 | 默认 `http://<主机>:<端口>/pallas/`（见 `pallas.toml` 监听配置） |
| 4. 在群里试一句 | 发 `牛牛帮助` 查看功能列表 |

## 网页控制台

控制台用来改配置、看日志、管插件，**不必改代码**。

| 想做什么 | 去这里 |
| --- | --- |
| 改画画 / 唱歌 / 远控地址 | 通用配置 → **外部服务地址与连通检测** |
| 改智能对话开关与地址 | 通用配置 → **智能对话与 AI 服务** |
| 改谁可以用某条口令 | 通用配置 → **命令权限** |
| 改口令冷却时间 | 通用配置 → **命令冷却** |
| 用社区共享接话素材 | 通用配置 → **社区共享接话库** |
| 多只牛同群不抢话 | 通用配置 → **多机协同** |
| 安装官方扩展插件 | 插件 → **官方扩展** |

配置会写入 `data/pallas_config/webui.json`，详见 [配置存储](../architecture/settings-storage.md)（运维向）。

## 常用口令

完整列表以群内 **`牛牛帮助`** 为准。core 功能示例：

| 口令 | 作用 |
| --- | --- |
| 牛牛帮助 | 功能总览与单插件说明 |
| 牛牛连通 / 牛牛网关 | 检测外部服务是否通畅 |
| 牛牛喝酒 / 牛牛醒一醒 | 饮酒与醒酒（影响接话等） |
| 牛牛轮盘 | 轮盘玩法 |
| 牛牛 | 打招呼（全员同响时多只牛都会回） |

各插件说明见 [插件索引](../plugins/README.md)。

## 常见问题

| 现象 | 先看 |
| --- | --- |
| 牛牛没反应 | [FAQ](../FAQ.md) · 是否被拉黑、冷却、或插件在本群关闭 |
| 智能对话不可用 | 通用配置里服务地址、是否开启；发 `牛牛连通` 看延迟 |
| 多只牛只回一只 | 通用配置 → 多机协同 / 全员同响口令 |
| 从旧版升级 | [3.0 迁移](../Migration-v3.md) |
