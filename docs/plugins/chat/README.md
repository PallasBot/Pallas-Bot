# chat（酒后聊天）

> **官方扩展**：`pallas-plugin-ai-media`（与 sing 同包，`uv sync --extra plugins-ai-media`）

牛牛**醉酒**时在群内 @ 它即可智能对话；须先「牛牛喝酒」。

## 用户命令

| 口令 / 触发 | 场景 | 说明 |
| --- | --- | --- |
| @牛牛 | 群内 | 醉酒时回复 |
| 牛牛 + 文本 | 群内 | 同上 |

## 命令权限

无独立口令权限（须处于醉酒状态）。

## 配置

与 [随时闲聊](./llm_chat/README.md) 共用 **通用配置 → 智能对话与 AI 服务** 总开关与服务地址。

## 排障

| 现象 | 处理 |
| --- | --- |
| 无回复 | 先喝酒；确认智能对话已开启且服务可达 |
| 冷却 | 群级冷却内可能静默 |
